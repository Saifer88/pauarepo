# Immagini del sito

Inserisci qui le immagini di Paola per il sito:

## File necessari

- `hero-bg.jpg` — Foto per lo sfondo della sezione hero (consigliata 1920x1080px, Paola in allenamento o in palestra)
- `paola-portrait.jpg` — Foto ritratto per la sezione "Chi Sono" (consigliata 800x1000px, formato verticale)

## Suggerimenti

- Usa foto ad alta risoluzione
- Le immagini hero funzionano meglio se leggermente scure o con buon contrasto
- Per il ritratto, un'immagine professionale con sfondo neutro è ideale
- Formati: JPG per foto, PNG per grafiche con trasparenza
